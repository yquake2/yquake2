# Function list generator

## About

These shell scripts generate the function lists and
prototypes for saving / loading function pointers
within the savegame system.

## Usage

First generate the data:

```
# cd to stuff/fngen
./fngenprotos.sh ../../src/game > proto.txt
./fngentables.sh ../../src/game > tables.txt
```

Then copy the contents of the files to the respective header
files in the savegame folder.

**NOTE**:: You need to remove `func` and `afterwaitfunc`
from the moveinfo.endfunc lists. This is a side-effect
of how the script identifies points of interest in code.

## Dependencies

These scripts use the following programs / commands:
* dirname
* awk
* grep

Windows users may need to download and install
one or more of these. Tested with Git Bash.

## Implementation details

**fntable.sh** and **fnproto.sh**: Helper scripts that generate
a table and a list of function prototypes for a specified
function pointer.

First `grep` uses function pointer assignments to get a list of
function names, then `awk` removes duplicate names and lastly
`sed` removes `NULL` entries and dresses up the function names
in the table entry/prototype context.

**fngentables.sh**: Writes to stdout the function tables for
all the function pointers in the edict_t struct.

**fngenprotos.sh**: Writes to stdout the prototypes for
all the functions referenced by the tables generated by fngentables.sh.
